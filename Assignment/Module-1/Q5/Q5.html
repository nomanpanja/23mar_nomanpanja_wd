5 : What are different types of lists in HTML?

- There are three types of list in HTML:
1. Ordered
2. Unordered
3. Description

Example :

1. Ordered List :
<ol>
   <li>Example</li>
</ol>

2. Unordered :
<ul>
   <li>Example</li>
</ul>

3. Description :
<dl>
   <dt>Example</dt>
   <dd>Example description</dd>
</dl>
